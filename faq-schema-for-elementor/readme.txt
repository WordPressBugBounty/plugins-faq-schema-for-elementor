== Description ==

FAQ Schema for Elementor adds a widget to Elementor that inserts FAQ schema (structured data) in JSON-LD format, without rendering anything on the front-end. Widget can be found under General.

Do not use more than one widget to a page, add multiple FAQs to the same widget instead.

Tested on Elementor 2.8.5 - 3.0.14

== Frequently Asked Questions ==

= Multiple FAQs =

You should only have one of this widget per page. If you have more than one FAQs, add them to the same widget.

= Nothing Shows on Front End =

That's the point of this plugin. There are plenty of widgets that display FAQs in accordions.
